# 跨境电商利润计算器

这是一个使用纯 HTML、CSS、JavaScript 构建的静态网页工具，可直接通过浏览器或 GitHub Pages 访问，无需后端服务器或构建步骤。

## 本地预览

直接双击打开 `index.html` 即可使用；也可以用任意静态文件服务器预览。

## GitHub Pages 部署

本项目入口文件位于仓库根目录的 `index.html`，样式和脚本使用相对路径加载，适合从 GitHub Pages 的仓库根目录发布。

在 GitHub 仓库中开启 Pages：

1. 打开仓库页面，进入 **Settings**。
2. 在左侧菜单点击 **Pages**。
3. 在 **Build and deployment** 区域，将 **Source** 选择为 **Deploy from a branch**。
4. 在 **Branch** 下拉框选择要发布的分支，例如 `main`，目录选择 `/ (root)`。
5. 点击 **Save**，等待 GitHub Pages 完成部署。

部署完成后，访问地址通常为：

```text
https://<你的 GitHub 用户名>.github.io/<仓库名>/
```

如果仓库配置了自定义域名，请使用 GitHub Pages 设置页显示的自定义域名访问。
